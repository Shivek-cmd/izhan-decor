# SEO Reference — Izhaan Decor Inc

## Rules (apply to every page, present and future)

| Field | Rule |
|-------|------|
| **Meta Title** | Max 70 characters. Format: `Primary Keyword \| Brand Name`. Must include the most important keyword for that page. |
| **Meta Description** | Max 155 characters. One or two sentences. Include the primary keyword naturally. End with a soft CTA (e.g. "Get a free quote."). |
| **Keywords** | 5–10 comma-separated terms. Start with the most specific (long-tail), end with broad terms. No stuffing — only terms genuinely relevant to that page. |
| **Canonical Link** | Always the full absolute URL with trailing slash. Use the production domain. Never a preview or staging URL. One canonical per page — no duplicates. |
